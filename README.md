# Portfolio Linux — CASANOVA Romann

Site statique (HTML/CSS/JS) inspiré d'une interface terminal / Linux.

## Lancer en local
- Ouvre `index.html` dans ton navigateur, ou
- Lance un serveur :
  - Python : `python -m http.server 8000`
  - Puis ouvre http://localhost:8000

## Photo
La photo est dans `assets/profile.jpg`.

## CV
La page `cv.html` affiche le CV via Google Drive (lien public du Google Sites).
Si tu préfères un CV local :
1) mets ton PDF dans `assets/cv.pdf`
2) dans `cv.html`, remplace l’iframe `src="https://drive.google.com/.../preview"` par `src="assets/cv.pdf"`.

## Contenu
Les textes (bio, projets, compétences, contacts) ont été repris du Google Sites fourni.
